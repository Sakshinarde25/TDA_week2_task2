# Student Grade Calculator

## Project Overview
This project demonstrates decision-making and looping concepts in Python
by calculating student grades based on marks.

## Objectives
- Use if-elif-else conditions
- Implement while loop for validation
- Create reusable functions

## Setup Instructions
1. Install Python
2. Clone repository
3. Run `python grade_calculator.py`

## Concepts Used
- Functions
- Conditional statements
- Loops
- Input validation

## What I Learned
- Writing decision-based programs
- Handling invalid user input
- Structuring Python programs
